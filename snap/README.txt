 This is a snap meant for new users and people to help them install lutris on a nvidia amd64 or intel64 bit system.
    To Install Follow these instructions in a terminal:
    cd /snap/lutrisinstaller/current/snap
    sudo chmod +x Installer.sh
    ./Installer.sh
    After Installer.sh:
    sudo chmod +x 'Lutris Install.sh'
    ./'Lutris Install.sh'
    Now Your Done :)
